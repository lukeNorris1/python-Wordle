Project:
Wordle program using python
	Program picks a word from the target.txt file and then prompts the user to guess the word giving them hints
		after each guess
USAGE:
	In a bash terminal run startServer.sh with the parameteres being the port of
		the server
	Example: startServer.sh  4000
		This connects the server to 192.168.1.2 using port 4000
	After the server is running run startclient.sh parameteres being the host connection and the port of
		the client
	Example: startClient.sh 192.1681.2 4000
	This connects the client to the server using the address 192.168.1.2 and the port 4000
